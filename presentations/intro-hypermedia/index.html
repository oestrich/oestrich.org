A bit about HTTP Verbs

GET

  • Retrieve a resource

should never change a resource

POST

  • Most generic verb
  • Can use it for anything

no semantic meaning

PUT

  • Update or Create a resouce, "Put this resource at this url"
  • Must be the entire resource, not partial

rails does this wrong

DELETE

  • Delete a resource

PATCH

  • Update a resource partially
  • Must be a diff

no current standard diff format, https://secure.designinghypermediaapis.com/nodes/much-ado-about-patch
